------------------------------------------
Buffs
------------------------------------------
orange = attack speed,
yellow = block chance,
blue = move speed,
cyan = dexterity,
red = crit chance,
dark red = crit damage,
grey = dodge chance,
pink = attack damage,
purple = damage multiplier,
brown = threat,
cross = minus stat X

--------------------------------------------
Conditions
------------------------------------------
orange = on hit or after X hits,
red = on crit or at X crit chance,
yellow = on block or at X block chance,
cross = below  stat X,
